============
Installation
============

Open Inwoner can be installed and run in several ways:

1. Run from :ref:`Python code <installation_development>` on your computer for
   development purposes
2. Run with :ref:`Docker-Compose <installation_docker_compose>` on your computer
   for private testing purposes
3. Deploy using :ref:`Ansible <installation_ansible>` for public testing and
   production purposes
